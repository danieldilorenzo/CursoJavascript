Identificadores

  • Podem começar com: Letra, $ ou _
  • Não podem começar com números
  • Podem conter letras ou números
  • Podem conter símbolos ou acentos
  • Não podem conter espaço
  • Não podem conter comandos como "print" ou "alert"

    • Tipos de dados

  • Number (infinity ou NAN - Not A Number)
  • String
  • Boolean
  • Null
  • Undefined
  • Object (array)
  • Function

    • Converter número para inteiro

    Number.parseInt(N)

    Converter número para float

    Number.parseFloat(N)

    Converter número para genérico

    Number(N)

    Converter para string

    String(n)
    n.toString

    Mostrar número com duas casas decimais

    n.toFixed(2)

    Mostrar um valor BR com real e duas casas decimais

    n.toLocaleString('pt-BR',{style: 'currency', currency: 'BRL'})
