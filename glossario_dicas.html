Identificadores

  • Podem começar com: Letra, $ ou _
  • Não podem começar com números
  • Podem conter letras ou números
  • Podem conter símbolos ou acentos
  • Não podem conter espaço
  • Não podem conter comandos como "print" ou "alert"

    Conjunção

    Quando duas afirmações estão corretas
    const a = 3;
    const b = -2;

    console.log(a > 0 || b > 0);
    // expected output: false

    Disjunção


    Se alguma das afirmações estiverem corretas
    const a = 3;
    const b = -2;

    console.log(a > 0 || b > 0);
    // expected output: true
